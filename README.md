# Algoritmos - Análise e Desenvolvimento de Sistemas (IFRN)

Este repositório armazena as atividades, exercícios e projetos práticos desenvolvidos durante a disciplina de **Algoritmos**, do curso de Análise e Desenvolvimento de Sistemas no Instituto Federal do Rio Grande do Norte (IFRN).

Todas as soluções e lógicas abordadas aqui são implementadas na linguagem **C++**.

## 📚 Conteúdos Abordados

A disciplina explora desde os conceitos fundamentais de lógica e matemática até a análise de eficiência computacional. Os principais tópicos estudados incluem:

* **Introdução aos Algoritmos:** Definições, características e representação matemática de problemas.
* **Análise de Eficiência:** Tempo de processamento, uso de memória e Notação Big-O.
* **Fundamentos de C++;**
* **Técnicas de Construção de Algoritmos:**
  * Força Bruta;
  * Tentativa e Erro (*Backtracking*);
  * Divisão e Conquista;
  * Programação Dinâmica;
  * Método Guloso.

## 💻 Tecnologia
* **Linguagem:** C++ (utilizando os padrões modernos da linguagem)
